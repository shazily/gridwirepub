What ships inside the boundary

ETags, Parquet export, OpenAPI specs, and connectors — after ingest passes the gate above.

PDF structure → data

Cheap structure pass, human approve, then full rows — with saved templates for recurring SFTP/folder PDFs.

Multi-tab parsing

Excel (.xlsx/.xls) and CSV with all tabs, headers, and computed formula values.

Field-mapping wizard

Auto-suggested API names, per-field types, PII flags, and masking (none / mask / hash).

Versioned REST API

Each sheet becomes a documented endpoint with limit/offset/fields and Bearer API-key auth.

Archive & delete audit

Take APIs offline, restore, or permanently delete with name confirmation and insert-only audit events.

Masking & hashing

Sensitive fields masked or hashed at serve time. Encryption in transit and at rest.

Multi-tenant

Organizations with role-based access and row-level isolation.

Connectors

SFTP, NFS, and watched folders via the companion worker.
