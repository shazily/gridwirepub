Turn any spreadsheet into a production API.

Gridwire converts Excel and CSV files into secure, documented REST APIs — with field mapping, versioning, masking, scoped API keys, multi-tenant organizations, automated connectors, and an admin console.

Get the Repo Self-host in minutes

Everything you need to ship data APIs

From messy multi-tab workbooks to clean, governed endpoints.

📊 Multi-tab parsing

Excel (.xlsx/.xls) and CSV with all tabs, headers, and computed formula values. Macros are detected and never executed.

🧭 Field-mapping wizard

Auto-suggested snake_case API names, per-field types, PII flags, and masking (none / mask / hash).

🔌 Generated REST API

Each sheet becomes a documented endpoint with limit/offset/fields and field filtering. Bearer API-key auth.

🕓 Versioning & diffs

Every upload is a version; the first is the baseline. Re-uploads are diffed for schema drift.

🔐 Masking & hashing

Sensitive fields are masked or hashed at serve time. Encryption in transit and at rest.

👥 Multi-tenant

Organizations with owner / admin / member / viewer roles and full row-level isolation.

📥 Connectors

SFTP, NFS / network shares, and watched folders via the companion worker — with test runs and logs.

🔔 Alerts

In-app feed plus email notifications for publishes, schema drift, and ingestion failures.

📖 Auto API docs

Per-dataset reference with cURL / JS snippets and a downloadable OpenAPI spec.

Deploy on-premise with Docker

Runs entirely on infrastructure you own — no external SaaS dependency.

git clone https://github.com/shazily/gridwirepub.git
cd gridwirepub
cp .env.example .env          # fill in your own values

docker compose up -d --build  # portal + ingestion worker

# portal on http://localhost:3000 — put your TLS reverse proxy in front
  1. Create an organization and sign in.
  2. Upload a spreadsheet and map fields in the wizard.
  3. Create an API key and call your new endpoint.
  4. Optionally add SFTP/NFS/folder connectors for automated ingestion.

Built for on-premise

Standard, vendor-neutral tooling. Deploy however your team already ships software.

🐳 Docker Compose

One host, one file. The recommended baseline for pilots and small teams.

☸️ Kubernetes

Reference manifests for a stateless, horizontally-scalable portal with rolling upgrades.

🔒 Your backend, your data

Postgres + Auth + REST run on your hardware. Air-gap friendly, no outbound calls.

🔑 Bring your own secrets

12-factor config via environment variables and your existing secrets manager.

💾 Backups & upgrades

All state in Postgres — standard pg_dump backups and migration-based upgrades.

🛡️ TLS everywhere

Terminate TLS at Nginx, Traefik, or Caddy. Example configs included.
